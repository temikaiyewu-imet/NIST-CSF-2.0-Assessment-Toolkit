# GRC Consulting Portfolio — NIST CSF 2.0 Assessment Toolkit

> A complete, professional-grade NIST Cybersecurity Framework 2.0 gap assessment
> toolkit built through consulting engagement on a fintech
> startup (PayOval Inc.). 
---

## About This Project

This project documents a full NIST CSF 2.0 consulting engagement from scoping
through to remediation planning. It was built to:
- Demonstrate end-to-end GRC consulting methodology
- Serve as a reusable toolkit for real client engagements
- Show practical application of the NIST CSF 2.0 framework

**Client:** PayOval Inc.
**Industry:** Fintech — Payment Processing
**Size:** 45 employees, seed-stage
**Finding:** Tier 1 (Partial) maturity across all 6 NIST CSF functions.

---

## What's Included

### 1. Assessment Toolkit (Excel)
[NIST_CSF_Assessment_Toolkit.xlsx](NIST_CSF_Assessment_Toolkit.xlsx)
- Gap assessment questionnaire for all 6 CSF functions (91 controls).
- YES / PARTIAL / NO dropdowns with evidence notes column.
- Auto-scoring dashboard with maturity tier calculation.
- Remediation roadmap with risk ratings and priority tiers (P1/P2/P3).

### 2. Audit Report Template (Word)
[NIST_CSF_Audit_Report_Template.docx](NIST_CSF_Audit_Report_Template.docx)
- 8-section professional report.
- Executive summary, scope and methodology, detailed findings per function.
- Risk register, 90-day remediation roadmap, policy requirements.
- Reassessment plan (retainer scope), consultant sign-off page.

### 3. Security Policy Pack (Word)
[PayOval_Policy_Pack.docx](PayOval_Policy_Pack.docx)
- SEC-POL-001: Cybersecurity Policy (Master)
- SEC-POL-002: Acceptable Use Policy
- SEC-POL-003: Password Policy
- SEC-POL-004: Incident Response Policy (with severity tiers and breach notification)
- SEC-POL-005: Data Classification Policy

## How to Use This

1. Download the Excel toolkit.
2. Navigate to each function tab (GV, ID, PR, DE, RS, RC).
3. Fill in YES / PARTIAL / NO for each control based on your client environment.
4. Check the Scoring Dashboard tab for maturity tier scores.
5. Use the Remediation Roadmap tab to prioritize fixes.
6. Populate the Word report template with your findings.
7. Customize the policy pack with client details.
